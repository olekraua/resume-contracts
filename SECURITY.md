# Security Policy

## Supported Versions

Only the `main` branch is supported with security updates.

## Reporting a Vulnerability

Do not open public issues for vulnerabilities.

1. Email the maintainer at `security@resume-platform.local` with:
- clear reproduction steps,
- impact assessment,
- suggested remediation (if available).
2. Include repository name and commit SHA.
3. Expect an initial response within 3 business days.

## Disclosure Process

1. Triage and reproduce.
2. Prepare and validate fix in a private branch.
3. Release patch and backport if needed.
4. Publish advisory after mitigation is available.
